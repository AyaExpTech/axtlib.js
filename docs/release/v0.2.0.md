## Canvas API - 拡張

各種図形を短いコードで描画できるように、以下のメソッドを追加しました。

- [`CanvasRenderingContext2D`](./../refs/CanvasRenderingContext2D/_.md)
    - `.prototype.axt_draw()` - 各種図形の描画 (汎用描画メソッド)
- [`OffscreenCanvasRenderingContext2D`](./../refs/CanvasRenderingContext2D/_.md)
    - `.prototype.axt_draw()` - 各種図形の描画 (汎用描画メソッド)
